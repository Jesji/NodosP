# Descripción
En este repositorio se crearon scripts para extraer, transforman y analizar los datos de los catálogos de NodosP publicados por el CENACE en el sitio (https://www.cenace.gob.mx/Paginas/SIM/NodosP.aspx). El proceso se compone de 3 partes: 
1. Extracción: Al correr el archivo "Downloader.py" se utiliza selenium para hacer webscraping de los catalogos de NodosP en formato .xlsx para el rango de años provisto. Cada vez que se corre este script, se crea la carpeta "DownloadCatalogos/", donde se guardaran los archivos descargados. Si previamente esta carpeta ya existía, es borrada y se crea una nueva.

2. Transformación: Al correr el script "Transform.py" se lee el archivo 
"Data/AGEEML_202512111134444.xlsx" que contiene el Catálogo Único de Claves de Áreas Geoestadísticas Estatales, Municipales y Localidades del INEGI, estos datos se agrupan por municipio y se exportan al archivo "Data/df_munis.parquet". Posteriormente se leen los archivos en el folder "DownloadCatalogos/", se concatenan todos los catalogos y se agrupan por clave de NodoP para producir un catálogo único de nodos, el cual se exporta al archivo "Data/df_cat_agg.parquet". Finalmente este catálogo único de nodosP se usa para estandarizar el dataframe de los catalogos concatenados, además se realiza un merge con los datos de municipios obtenidos del INEGI para asignar las coordenadas del municipio en que localiza cada NodoP. Este dataframe, que es el que usará para el análisis, se exporta en el archivo "Data/df_nodosp.parquet".

3. Análisis: Al correr el script "Analysis.py" se lee el archivo "Data/df_nodosp.parquet" y se realiza una serie de agrupaciones a los datos para analizar la variación del total de NodosP en cada Centro de Control Regional y por nivel de Tensión. Se agrupan por CCR y por año para observar como ha variado el número de nodosP y se produce el gráfico "Graphics/NodosP-CCR.png". Posteriormente se agrupan los NodosP por año y por nivel de tensión y se producen las figuras "Graphics/nodosp-tension.png" y "Graphics/NodosP-ccr-tension.png". Finalmente se representa la localización los NodosP en el mapa de acuerdo a las coordenadas del municipio en que se encuentran, se produce el gráfico "Graphics/MapaInter.html".
